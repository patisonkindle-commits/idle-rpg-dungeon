# Idle RPG Dungeon

A browser-based idle dungeon crawler built with Phaser 3 + Vite.

## How to Play

- Open the game in your browser
- Your party auto-adventures through procedurally generated dungeons
- Heroes fight monsters, find treasure, rest, and level up
- Clear all rooms to advance to the next level
- Bosses appear every 5 levels!

## Tech Stack

- Phaser 3 - Game engine
- Vite - Build tool
- TypeScript - Type safety

## Build & Run

```bash
npm install
npm run build
npm run preview
```
